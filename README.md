## Introduction

LightBrush is designed for speed, simplicity, and professional-grade features.  

It is suitable for:  

- Digital painting  
- Concept art  
- Illustrations  
- Manga and comics  
- UI and UX sketches  

LightBrush works smoothly on low-end hardware and supports both raster and vector workflows.  

LightBrush is aimed at beginners, professionals, and hobbyists alike. It provides a stable, intuitive, and responsive interface for digital artists.  

The main goals of LightBrush are:  

- Lightweight performance  
- Professional tools  
- Open-source collaboration  
- Multi-platform support  

With LightBrush, you can create stunning digital artwork, sketches, illustrations, and concept art efficiently and comfortably.

## Features

### General Features

- Cross-platform: Windows, macOS, Linux  
- High DPI support  
- Customizable workspace  
- Multilingual interface  
- Open-source and free  

### Painting Features

- Multi-layer painting with blending modes  
- Advanced brush engine with pressure sensitivity  
- Stabilizers for smooth strokes  
- Dynamic brush presets  
- Real-time brush previews  

### Color Features

- RGB, CMYK, and HSL color models  
- Color wheel and palettes  
- Gradient editor  
- Swatch management  
- Color history  

### Advanced Features

- Non-destructive editing  
- Layer groups and masks  
- Transform tools  
- Selection tools  
- Vector shapes and paths  

### Productivity Features

- Customizable keyboard shortcuts  
- Drag and drop support  
- Export in PNG, JPEG, PSD, and SVG  
- Unlimited undo and redo  
- Quick navigation bookmarks  

LightBrush aims to provide a smooth workflow for all digital artists while keeping system requirements minimal.

## Installation

### Windows

- Download the installer from the official LightBrush website  
- Run the installer and follow on-screen instructions  
- Launch LightBrush from the Start Menu  

### macOS

- Download the DMG file from the official website  
- Drag LightBrush into the Applications folder  
- Open LightBrush from the Applications folder  

### Linux

- Download the AppImage from the official website  
- Make it executable and open LightBrush  
- LightBrush works on all modern Linux distributions

## Quick Start

- Open LightBrush  
- Create a new canvas from the File menu  
- Select a brush from the Toolbar  
- Start painting on the canvas  
- Organize your work using layers  
- Save your project regularly  

LightBrush offers an intuitive interface that allows you to start painting immediately without a steep learning curve.  

---

## Interface Overview

- Toolbar: Left side, contains brushes, erasers, and selection tools  
- Layers Panel: Right side, manage layers, masks, and blending modes  
- Color Panel: Right side, pick colors, gradients, and swatches  
- Menu Bar: Top, access file, edit, view, and tools  
- Canvas: Center, the area for drawing and painting  

Each component of the interface can be customized according to user preference. Panels can be moved, hidden, or resized to optimize workflow.

## FAQ

**Q: Can I import Krita brushes?**  
A: Yes, LightBrush supports Krita brush formats.  

**Q: Does LightBrush support PSD?**  
A: Yes, both import and export are supported.  

**Q: Is tablet supported?**  
A: Yes, Wacom, Huion, XP-Pen, and other tablets are fully supported.  

**Q: Is LightBrush free?**  
A: Yes, it is fully free and open-source.  

**Q: Can I work in CMYK?**  
A: Yes, CMYK mode is available for print projects.
